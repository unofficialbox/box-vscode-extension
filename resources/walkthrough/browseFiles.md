## Browse Files and Folders

The **All Files** view in the Box sidebar displays your entire Box file tree with lazy loading and "Load More" pagination for large directories.

From the tree view you can:
- **Create folders** at any level
- **Delete items** with confirmation
- **Copy IDs** to clipboard
- **Filter** files by name
- **Download** files and folders
- **Right-click** for context menu actions including Box UI Elements
